Default settings project Django
===============================

This project is based django version 2.2

Elements
--------
- travis.yml
- gitlab-ci.yml

Install elements production :
-----------------------------
- django==2.2
- psycopg2

    pip install -r requiments.txt

------------------------------

- flake8
- sphinx
- pytest
- pytest-cov
- coverage
- ipython
- django-debug-toolbar

.. code-block:: bash

    pip install -r test_requirements.txt
